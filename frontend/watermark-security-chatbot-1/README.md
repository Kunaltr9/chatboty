# Watermark Security Intelligence Chatbot

## Overview
The Watermark Security Intelligence Chatbot is a modern, interactive application designed to analyze Apache logs, detect anomalies, and provide security insights for watermarking and Data Leak Prevention (DLP) systems. Built with React and Tailwind CSS, this chatbot offers a sleek user interface with a focus on user experience and accessibility.

## Features
- **Log Analysis**: Detect security threats, performance issues, and anomalies in log data.
- **Interactive UI**: A modern design with a deep black background and purple-blue gradient accents.
- **Responsive Design**: Optimized for various devices, ensuring a seamless experience on desktops, tablets, and mobile devices.
- **Real-time Insights**: Get immediate feedback and recommendations based on log analysis.

## Technology Stack
- **Frontend**: React, Tailwind CSS
- **Backend**: Groq API for chat functionality
- **Data Handling**: Custom hooks and services for log analysis
- **Visualization**: Interactive components for displaying results and insights

## Installation
To set up the project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/watermark-security-chatbot.git
   cd watermark-security-chatbot
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the application**:
   ```bash
   npm run dev
   ```

4. **Open your browser** and navigate to `http://localhost:3000` to view the application.

## Usage
- Enter your Groq API key in the provided input field to enable chat functionality.
- Use the sidebar for quick access to various analysis options.
- Interact with the chatbot in the chat panel to receive insights and recommendations based on your log data.
